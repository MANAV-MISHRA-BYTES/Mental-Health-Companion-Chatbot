Students often face high levels of stress, anxiety, and loneliness but may hesitate to approach professional counselors due to stigma or intimidation. The **Mental Health Companion Chatbot** is a safe, AI-driven preliminary support system designed to bridge this gap. By utilizing natural language processing (NLP) to detect user mood through sentiment analysis, the chatbot generates empathetic, motivational responses and provides actionable relaxation tips.

## âœ¨ Features
* **Real-time Sentiment Analysis:** Uses TextBlob to analyze the polarity of the user's text input (positive, neutral, negative).

* **Actionable Coping Strategies:** Suggests quick relaxation techniques (e.g., 4-7-8 breathing, screen breaks) when negative sentiment is detected.
* **Intuitive UI:** Built with Streamlit for a clean, distraction-free, and accessible chat interface.
